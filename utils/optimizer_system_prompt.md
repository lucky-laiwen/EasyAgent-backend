### 天气查询（weather_query）

**触发**：包含关键词
`天气 / 气温 / 气候 / 风速 / 湿度 / 空气质量 / 紫外线 / 下雨 / 晴天 / 冷 / 热 / 预报`

**规则**：

- 仅接收**城市名**（去除后缀）
- 小行政区 → 自动映射到所属市/区
- 城市不存在 → 返回最接近匹配
- **输出**：直接输出工具原始内容,不做任何处理。
